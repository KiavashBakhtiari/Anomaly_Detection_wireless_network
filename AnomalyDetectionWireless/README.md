Anomaly Detection in Wireless Networks Using Machine Learning
This project implements ML-based anomaly detection for wireless networks using scikit-learn.

\# Anomaly Detection in Wireless Networks



این پروژه شبیه‌سازی تشخیص آنومالی در شبکه‌های بی‌سیم با استفاده از یادگیری ماشین است.



\## ویژگی‌ها

\- تولید داده‌ی شبیه‌سازی شده شبکه

\- پیش‌پردازش و نرمال‌سازی داده‌ها

\- تشخیص آنومالی با Random Forest و Isolation Forest

\- ارزیابی مدل و نمایش نتایج
